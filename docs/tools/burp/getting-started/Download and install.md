# 下载安装

下载安装方式有以下两种，

1. JAR文件，依赖Java运行环境，需要配置Java环境，支持版本为9-14。
2. 本地安装包，支持Linux（64-bit）、MacOS (ARM/M1)、MacOS (Intel)和Windows (64-bit)。

!> 建议前往官方网站下载原始安装包。<br />
